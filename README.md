## About The Project
This project showcases a clean, dark-themed product designer portfolio, integrating advanced animation effects, smooth scrolling, and a modern responsive layout.  
The site highlights key professional info, projects, and personal branding in an interactive and accessible way.

---

## Features
- Modern, minimalist portfolio layout with dark theme
- Smooth scrolling powered by Locomotive Scroll
- Animated text and UI reveal effects using GSAP (GreenSock Animation Platform)
- Responsive navigation and hero section
- Interactive mouse follower animation
- Sectioned portfolio display with project highlights and about/contact info
- Social/profile links and easy contact functionality
- Optimized for desktop and mobile viewing

## Skills & Tools Used
| Category        | Details                                                                    |
| --------------  | -------------------------------------------------------------------------- |
| **HTML5**       | Semantic structure, accessibility enhancements                            |
| **CSS3**        | Flexbox, layout, animations, responsive design, custom scroll effects      |
| **JavaScript**  | DOM manipulation, event handling, animation logic                         |
| **GSAP**        | Timeline and reveal animations, mouse follower                            |
| **Locomotive Scroll** | Smooth custom scroll experience and parallax effects                |
| **Remixicon**   | Icon library for navigation and branding                                  |
